      About SecurePass Vault

      SecurePass Vault is a modern password tool built using pure HTML, CSS, and JavaScript. It demonstrates how to generate, check, and manage passwords in a secure, responsive, and accessible way — all in your browser.

      • Strong password generator
      • Real-time strength checker
      • Local storage vault
      • Breach check via HIBP API
      • Dark mode support

      ⚠️ Disclaimer: This demo uses client-side storage. Do not store sensitive production passwords here.
